# تقرير تطوير منصة تحليل السوق السعودي (تداول)

## نظرة عامة

تم تطوير منصة متكاملة لتحليل السوق السعودي (تداول) تتضمن مجموعة من الوحدات البرمجية المترابطة التي تعمل معًا لجمع ومعالجة وتحليل وعرض البيانات المالية للشركات المدرجة في السوق السعودي. تهدف المنصة إلى مساعدة المستثمرين في اتخاذ قرارات استثمارية مدروسة بناءً على تحليلات فنية وأساسية شاملة.

## المكونات الرئيسية المطورة

### 1. وحدة جمع البيانات (Data Collection)

تم تطوير وحدة متخصصة لجمع البيانات المالية من مصادر متعددة:

- **`financial_data_collector.py`**: وحدة لجمع البيانات المالية للشركات السعودية من مصادر مختلفة
  - جمع بيانات القوائم المالية (الميزانية العمومية، قائمة الدخل، التدفقات النقدية)
  - جمع البيانات الأساسية للشركات (القطاع، الصناعة، عدد الأسهم، إلخ)
  - تنفيذ آلية للتخزين المؤقت (caching) لتحسين الأداء وتقليل عدد الاستدعاءات للواجهات البرمجية

- **`market_data_collector.py`**: وحدة لجمع بيانات السوق
  - جمع بيانات التداول اليومية (السعر، الحجم، إلخ)
  - جمع بيانات المؤشرات والقطاعات

- **`enhanced_data_collector.py`**: وحدة لجمع بيانات معززة من مصادر متعددة
  - جمع وتحليل بيانات الأخبار من مصادر مختلفة
  - تحليل المشاعر من وسائل التواصل الاجتماعي
  - جمع توصيات وأهداف الأسعار من المحللين
  - جمع بيانات الاقتصاد الكلي المؤثرة على السوق

### 2. وحدة معالجة البيانات (Data Processing)

تم تطوير وحدة متخصصة لمعالجة وتنظيف وتحويل البيانات المالية:

- **`financial_data_processor.py`**: وحدة لمعالجة وتطبيع البيانات المالية
  - تطبيع القوائم المالية (Normalization) لتسهيل المقارنة بين الشركات
  - حساب معدلات النمو (Growth Rates) للمؤشرات المالية المختلفة
  - حساب قيم الـ TTM (Trailing Twelve Months) من البيانات الربعية
  - توحيد البيانات (Standardization) للمقارنة مع متوسطات القطاع
  - حساب النسب المالية المختلفة

### 3. وحدة تخزين البيانات (Data Storage)

تم تطوير وحدة لتخزين واسترجاع البيانات بكفاءة:

- **`database_manager.py`**: وحدة لإدارة قاعدة البيانات
  - إنشاء وإدارة قاعدة بيانات SQLite
  - تخزين بيانات الشركات والقوائم المالية وبيانات السوق
  - توفير واجهة برمجية سهلة للوصول إلى البيانات واستعلامها
  - دعم البحث والتصفية المتقدمة للبيانات

### 4. وحدة التحليل (Analysis)

تم تطوير وحدات للتحليل الفني والأساسي ونماذج التعلم الآلي:

- **التحليل الفني**:
  - `indicators.py`: حساب المؤشرات الفنية (المتوسطات المتحركة، RSI، MACD، إلخ)
  - `analyzer.py`: تحليل الأنماط الفنية واكتشاف إشارات التداول

- **التحليل الأساسي**:
  - `ratios.py`: حساب النسب المالية المختلفة
  - `statements.py`: تحليل القوائم المالية
  - `valuation.py`: تقييم الشركات باستخدام نماذج مختلفة

- **نماذج التعلم الآلي**:
  - `ml_recommendation_model.py`: تنفيذ نماذج التعلم الآلي لتحسين دقة التوصيات
  - تطبيق خوارزميات Random Forest وGradient Boosting وNeural Network
  - اختيار الميزات وتحسين المعلمات تلقائيًا
  - نموذج تجميعي (Ensemble) يجمع بين توقعات النماذج المختلفة

### 5. وحدة التوصيات (Recommendation)

تم تطوير محرك للتوصيات يجمع بين نتائج التحليل الفني والأساسي ونماذج التعلم الآلي:

- **`engine.py`**: محرك التوصيات الأساسي الذي يولد توصيات الشراء والبيع والاحتفاظ
- **`enhanced_engine.py`**: محرك توصيات معزز يدمج نماذج التعلم الآلي مع البيانات المعززة
- **`screener.py`**: أداة لفرز الأسهم بناءً على معايير محددة
- **`portfolio.py`**: إدارة المحافظ الاستثمارية وتحليل الأداء
- **`alerts.py`**: نظام التنبيهات لإشعار المستخدمين بالفرص الاستثمارية

### 6. واجهة المستخدم (Frontend)

تم تطوير مكونات واجهة المستخدم باستخدام React و Material UI:

- **مكونات التوصيات**:
  - `RecommendationCard.js`: بطاقة عرض التوصية الفردية
  - `RecommendationsList.js`: قائمة التوصيات مع إمكانية التصفية والبحث
  - `RecommendationDetailsDialog.js`: نافذة تفاصيل التوصية
  
- **مكونات مشتركة**:
  - `TabPanel.js`: مكون للتبويبات المستخدمة في واجهة المستخدم

## الميزات الرئيسية المنفذة

### 1. جمع ومعالجة البيانات

- جمع البيانات المالية من مصادر متعددة
- معالجة وتنظيف البيانات لضمان الدقة والاتساق
- تخزين البيانات في قاعدة بيانات منظمة
- تحديث البيانات بشكل دوري

### 2. التحليل الفني والأساسي

- حساب المؤشرات الفنية المختلفة
- تحليل الأنماط الفنية واكتشاف إشارات التداول
- تحليل القوائم المالية وحساب النسب المالية
- تقييم الشركات باستخدام نماذج مختلفة
- تحليل المشاعر من الأخبار ووسائل التواصل الاجتماعي
- دمج بيانات المحللين وتوصياتهم في التحليل

### 3. توليد التوصيات المعززة بالتعلم الآلي

- توليد توصيات الشراء والبيع والاحتفاظ باستخدام نماذج التعلم الآلي
- تحديد أهداف السعر ومستويات وقف الخسارة بدقة أعلى
- تقييم نسبة المخاطرة/العائد لكل توصية
- تصنيف التوصيات حسب درجة الثقة المستندة إلى نماذج متعددة
- تحديد أهم العوامل المؤثرة في كل توصية
- تقليل الإشارات الكاذبة باستخدام نماذج التجميع (Ensemble)

### 4. واجهة المستخدم

- عرض التوصيات بطريقة سهلة الاستخدام
- تصفية وبحث التوصيات حسب معايير مختلفة
- عرض تفاصيل التوصية مع التحليلات الداعمة
- إمكانية حفظ التوصيات وإضافتها إلى المحفظة

## التقنيات المستخدمة

### الواجهة الخلفية (Backend)

- **لغة البرمجة**: Python, PHP (Laravel)
- **قاعدة البيانات**: SQLite
- **مكتبات التحليل**: Pandas, NumPy
- **مكتبات التحليل الفني**: TA-Lib

### واجهة المستخدم (Frontend)

- **إطار العمل**: React
- **مكتبة واجهة المستخدم**: Material UI
- **إدارة الحالة**: React Context API
- **التعريب**: i18next

   - دمج توصيات المحللين وأهداف الأسعار
   - جمع بيانات الاقتصاد الكلي المؤثرة على السوق

## الخلاصة

تم تطوير منصة متكاملة لتحليل السوق السعودي (تداول) تجمع بين التحليل الفني والأساسي لتوليد توصيات استثمارية دقيقة. المنصة قادرة على جمع ومعالجة وتحليل البيانات المالية للشركات المدرجة في السوق السعودي، وتقديم توصيات مدعومة بتحليلات شاملة. واجهة المستخدم سهلة الاستخدام وتوفر تجربة سلسة للمستخدمين.

المشروع يمثل خطوة مهمة نحو توفير أدوات تحليلية متقدمة للمستثمرين في السوق السعودي، مما يساعدهم على اتخاذ قرارات استثمارية مدروسة بناءً على بيانات وتحليلات موثوقة.
