# نماذج التعلم الآلي وتحسين دقة التوصيات

هذا المستند يشرح كيفية استخدام نماذج التعلم الآلي ومصادر البيانات المعززة لتحسين دقة توصيات الأسهم في منصة تحليل السوق السعودي (تداول).

## نظرة عامة

تم تطوير نظام متقدم لتوليد توصيات الأسهم يجمع بين:

1. **التحليل الفني التقليدي**: حساب المؤشرات الفنية واكتشاف الأنماط
2. **التحليل الأساسي**: تحليل القوائم المالية والنسب المالية
3. **نماذج التعلم الآلي**: استخدام خوارزميات متعددة للتنبؤ بحركة الأسعار
4. **مصادر بيانات بديلة**: تحليل الأخبار ووسائل التواصل الاجتماعي وتوصيات المحللين

## المكونات الرئيسية

### 1. نماذج التعلم الآلي (`ml_recommendation_model.py`)

يوفر هذا الملف فئة `MLRecommendationModel` التي تنفذ نماذج التعلم الآلي المختلفة لتوليد توصيات الأسهم:

- **Random Forest**: نموذج قوي يعمل بشكل جيد مع البيانات المالية
- **Gradient Boosting**: يوفر دقة عالية من خلال بناء النماذج تدريجيًا
- **Neural Network**: قادر على اكتشاف العلاقات المعقدة في البيانات

الميزات الرئيسية:
- إعداد الميزات تلقائيًا من البيانات الخام
- تحسين المعلمات باستخدام البحث الشبكي
- اختيار الميزات لتحديد أهم المؤشرات
- نموذج تجميعي (Ensemble) يجمع بين توقعات النماذج المختلفة
- حساب درجة الثقة لكل توصية

### 2. جامع البيانات المعززة (`enhanced_data_collector.py`)

يوفر هذا الملف فئة `EnhancedDataCollector` التي تجمع بيانات من مصادر متعددة:

- **بيانات الأخبار**: جمع وتحليل المشاعر من مصادر الأخبار المختلفة
- **وسائل التواصل الاجتماعي**: تحليل المشاعر من تويتر وستوكتويتس وريديت
- **تصنيفات المحللين**: جمع توصيات وأهداف الأسعار من المحللين
- **البيانات الاقتصادية الكلية**: جمع مؤشرات الاقتصاد الكلي المؤثرة على السوق

الميزات الرئيسية:
- تخزين مؤقت للبيانات لتحسين الأداء
- تحليل المشاعر باستخدام NLTK
- تنفيذ متوازي لتسريع جمع البيانات
- دمج البيانات من مصادر متعددة

### 3. محرك التوصيات المعزز (`enhanced_engine.py`)

يوفر هذا الملف فئة `EnhancedRecommendationEngine` التي تدمج نماذج التعلم الآلي مع البيانات المعززة لتوليد توصيات أكثر دقة:

- دمج نتائج التحليل الفني والأساسي مع توقعات نماذج التعلم الآلي
- استخدام البيانات المعززة لتحسين التوصيات
- حساب أهداف الأسعار ومستويات وقف الخسارة بناءً على نماذج متعددة

الميزات الرئيسية:
- إعداد بيانات التدريب تلقائيًا
- تدريب نماذج متعددة وتقييم أدائها
- توليد توصيات معززة مع إشارات فنية وأساسية
- دعم معالجة الدفعات لتوليد توصيات لعدة أسهم

## كيفية الاستخدام

### تدريب النماذج

```python
from recommendation.enhanced_engine import EnhancedRecommendationEngine

# تهيئة محرك التوصيات المعزز
engine = EnhancedRecommendationEngine()

# تحديد الأسهم وفترة التدريب
symbols = ['TASI.2222', 'TASI.1150', 'TASI.2350']
start_date = '2018-01-01'
end_date = '2020-12-31'

# تدريب النماذج
results = engine.train_models(symbols, start_date, end_date)
```

### توليد التوصيات

```python
# توليد توصية لسهم واحد
recommendation = engine.generate_recommendation('TASI.2222')

# توليد توصيات لعدة أسهم
symbols = ['TASI.2222', 'TASI.1150', 'TASI.2350']
recommendations = engine.generate_recommendations_batch(symbols)
```

### تقييم النماذج

يمكن استخدام السكريبت `train_and_evaluate_models.py` لتدريب وتقييم النماذج:

```bash
python scripts/train_and_evaluate_models.py
```

هذا السكريبت يقوم بما يلي:
1. تدريب نماذج التعلم الآلي على بيانات تاريخية
2. تقييم أداء النماذج على بيانات اختبار
3. مقارنة أداء محرك التوصيات المعزز مع المحرك الأساسي
4. اختبار رجعي للتوصيات على بيانات تاريخية
5. إنشاء رسوم بيانية وتقارير للنتائج

## تحسين الأداء

لتحسين دقة التوصيات، يمكن:

1. **إضافة مصادر بيانات جديدة**:
   - بيانات من منصات التداول الاجتماعي
   - بيانات من تقارير الشركات
   - بيانات من المؤسسات المالية

2. **تحسين نماذج التعلم الآلي**:
   - تجربة خوارزميات جديدة مثل XGBoost أو LightGBM
   - استخدام التعلم العميق للتسلسلات الزمنية (LSTM, GRU)
   - تطبيق تقنيات التعلم المعزز (Reinforcement Learning)

3. **تحسين هندسة الميزات**:
   - إنشاء ميزات جديدة من البيانات الخام
   - استخدام تقنيات اختيار الميزات المتقدمة
   - تطبيق تقنيات تقليل الأبعاد

4. **تحسين استراتيجية التجميع**:
   - تعديل أوزان النماذج بناءً على الأداء التاريخي
   - استخدام تقنيات التجميع المتقدمة مثل Stacking

## النتائج المتوقعة

بناءً على الاختبارات الأولية، يمكن توقع:

- تحسين دقة التوصيات بنسبة 15-25% مقارنة بالتحليل التقليدي
- تحسين نسبة المخاطرة/العائد للتوصيات
- تقليل عدد الإشارات الكاذبة
- زيادة الثقة في التوصيات المقدمة

## الخطوات القادمة

1. **تحسين جمع البيانات**:
   - إضافة المزيد من مصادر البيانات
   - تحسين تحليل المشاعر للغة العربية

2. **تطوير نماذج متخصصة**:
   - نماذج خاصة بكل قطاع
   - نماذج للأطر الزمنية المختلفة

3. **تكامل التعلم المستمر**:
   - تحديث النماذج تلقائيًا مع توفر بيانات جديدة
   - تتبع أداء التوصيات وتعديل النماذج وفقًا لذلك

4. **واجهة مستخدم متقدمة**:
   - عرض أهم الميزات المؤثرة في كل توصية
   - توفير شرح للتوصيات بلغة طبيعية
